Detailed Description

template<typename T>
class Pothos::Util::RingDeque< T >

RingDeque is a template deque using an underlying vector. The API of RingDeque is very similar to the std::deque<T>. RingDeque is here because I wanted the efficiency of boost::circular_buffer without the boost requirement. The ring deque does not have a specific capacity limit.
